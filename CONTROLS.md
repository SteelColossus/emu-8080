## Space Invaders
| Key           | Action           |
|---------------|------------------|
| Right Shift   | Insert Coin      |
| Enter         | Player 1 Start   |
| Backspace     | Player 2 Start   |
| Space         | Player 1/2 Shoot |
| Left          | Player 1/2 Left  |
| Right         | Player 1/2 Right |
| Backquote (`) | Tilt Machine     |
